실제로는 깜빡임이 없지만 영상으로만 DotMatrix의 깜빡임이 존재합니다.</br>

- 1을 누르면 게임 시작 </br>
![vid](vid/press1tostart.gif)
</br> 오른쪽의 CLCD에서 게임시작버튼 안내문구 출력
</br> 1을 누르면 왼쪽의 DotMatrix에서 게임 출력 

- 게임플레이 </br>
![vid](vid/score.gif)
</br> 우주선을 조작하여 게임을 플레이하고 CLCD에 점수 출력

- 게임오버 </br>
![vid](vid/restart.gif)
</br> 게임오버되면 재시작버튼 안내문구 CLCD에 출력
